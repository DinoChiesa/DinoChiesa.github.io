Notes

  1. This tool uses EcmaScript v9 for promises and spread syntax, webcrypto for key derivation, generation, signing and verification, and Blob for file download.

    It will run only on modern, current browsers.

  2. The page also uses Bootstrap 4.0 for styling, the Open Iconic fonts for button symbols, and jQuery for interactivity.

  3. Currently it downloads just the privatekey PEM, but... we could easily modify it to download a JSON file containing the pem string as well as metadata about the key: when it was generated, the app name, and maybe other information.
